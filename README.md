This personal project is to keep a track of the stocks and shares that I own and compare current prices of stocks with historical data as well as calculating the total value of all my holdings.

My aim is to have the following set of data frames where I can I record data on my portfolio:
    #transactions_df   = this will contain my raw trade history (I will be adding bought and sold shares into this data frame)
    #prices_df         = this will contain the latest prices for my stocks and shares (API)
    #positions_df      = this will contain my aggregated holdings 
    #portfolio_df      = this will contain gains, value, weights
    #price_history_df  = this will contain a list of historical prices

I will then use this data to create line charts which will forecast my financial position over the next year for each of the stocks I've purchased.
